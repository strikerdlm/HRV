# Highlights

- We introduce Operational Performance Indicators (OPI) for aviation and UAS operators.
- OPI fuses SAFTE, HRV, and Multiple Resource Theory-derived task-calibrated weights.
- Per-task weights are specified for 10 manned-aviation and 7 UAS task categories.
- A Warm-type vigilance model and Chen-type latency penalty cover UAS teleoperation.
- Open-source reference implementation is distributed under the MIT license.
